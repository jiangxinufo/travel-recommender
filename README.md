# 县镇旅行随机推荐器

一个打开即用的静态网页，用来随机推荐县城和乡镇级旅游目的地。

## 使用

直接用浏览器打开 `index.html`，点击“随机推荐”即可。

## 功能

- 按省份、区域、旅行偏好、季节筛选目的地
- 默认按当前日期识别季节，并优先推荐当前季节适宜目的地
- 可选择优先避开过度拥挤目的地
- 每个目的地包含行政层级、适合天数、推荐季节、旅游资源、推介理由、轻松玩法、自驾游和大交通建议
- 中国大陆 31 个省级行政区各维护 10 个目的地，共 310 个
- 支持生成分享图、复制分享文案，并提供贡献目的地入口
- 每个目的地支持代表图；默认优先尝试 Wiki/Wikimedia 开放图源，失败时再使用配置中的备用 `image`，仍失败则使用稳定的本地主题图

## 数据维护

目的地数据维护在 `data/destinations.config.js`，不再放在 `app.js`。

`contributionUrl` 是贡献目的地入口，默认指向腾讯文档首页。上线前建议替换成你自己的问卷星或腾讯文档收集表链接。

每个省级区域使用一个配置块：

```js
{
  code: "liaoning",
  name: "辽宁省",
  region: "东北",
  transport: transport("辽宁"),
  destinations: [
    d("小市镇", "本溪市本溪满族自治县", "乡镇级", ["山水洗肺", "温泉放松"], ALL, true, "本溪水洞、太子河、小市羊汤和汤沟温泉", "从沈阳周末可达，洞穴山水温泉组合完整。")
  ]
}
```

新增或替换目的地时，只需要在对应省份的 `destinations` 数组里维护条目。

如果要为 Wiki 未命中的目的地增加备用照片，可以在条目中增加：

```js
image: "https://example.com/photo.jpg",
imageCredit: "图片来源：景区官网/摄影师/开放图库"
```

默认自动图源已经回到 Wiki/Wikimedia；备用照片建议使用景区官网授权图、政府文旅图、官方公众号图床或自有 CDN 图片，并保留来源说明。
