# 贡献目的地表单模板

> 用途：收集用户推荐的县城、乡镇、村落级旅游目的地，方便后续审核后加入目的地配置库。

## 基础信息

1. 目的地名称：
   示例：小市镇、沙溪镇、宏村镇、丙中洛镇、篁岭村

2. 所在省份：
   示例：辽宁省

3. 所在城市/县区：
   示例：本溪市本溪满族自治县

4. 行政层级：
   可选：县级 / 县级市 / 乡镇级 / 街道级 / 村落级 / 景区片区

5. 你是否去过这里：
   可选：去过 / 计划去 / 本地人推荐 / 朋友推荐

## 旅行适配

6. 适合季节：
   可多选：春季 / 夏季 / 秋季 / 冬季 / 四季皆宜

7. 建议游玩天数：
   示例：1天、2天、2-3天、3-4天

8. 旅行节奏：
   可选：轻松 / 慢节奏 / 中等 / 轻户外 / 深度游

9. 适合的放松感：
   可多选：古镇慢逛 / 山水洗肺 / 海滨放空 / 海岛放空 / 湿地观鸟 / 森林疗愈 / 温泉放松 / 民族风情 / 文化体验 / 摄影写生 / 亲水避暑 / 自驾公路 / 乡村美食 / 低强度放空

10. 是否适合避开过度拥挤：
    可选：是 / 否 / 分季节

## 旅游资源

11. 核心旅游资源：
    请用 1-3 句话说明这里最值得看的内容。

    示例：
    本溪小市镇周边有本溪水洞、太子河、小市羊汤和汤沟温泉，适合周末做山水、洞穴、美食和温泉组合游。

12. 代表性景点或体验：
    示例：
    - 本溪水洞
    - 小市羊汤
    - 关门山
    - 汤沟温泉

13. 本地特色：
    示例：地方小吃、非遗、民宿、赶集、茶园、渔港、温泉、古建、民族节庆等。

## 推介理由

14. 你推荐它的理由：
    请说明为什么它适合节假日放松。

    示例：
    从沈阳出发周末可达，不需要长途奔波；洞穴、山水、温泉和小城美食都能覆盖，适合想短暂离开城市节奏的人。

15. 更适合什么人：
    可多选：情侣 / 朋友 / 亲子 / 独自旅行 / 父母长辈 / 摄影爱好者 / 自驾游 / 徒步爱好者 / 只想放空的人

16. 不太适合什么情况：
    示例：旺季停车难、山路较多、公共交通不方便、雨天体验打折、海拔较高等。

## 交通信息

17. 自驾游建议：
    请写明从附近大城市或省会出发的大致路线、停车提醒、是否适合自驾。

    示例：
    沈阳方向可走丹阜高速进入本溪县，再导航至小市镇或本溪水洞。节假日建议早出发，返程避开傍晚高峰。

18. 大交通建议：
    请写明可抵达的机场、高铁站、火车站、客运站，以及最后一段如何到目的地。

    示例：
    可先到沈阳桃仙国际机场、沈阳站/沈阳北站，或到本溪站，再换乘县内客运、景区直通车或网约车。

## 推荐玩法

19. 推荐行程：
    请给一个简单行程。

    示例：
    第一天上午到小市镇吃午饭，下午游本溪水洞，晚上住县城或汤沟方向；第二天按体力选择关门山、绿石谷或温泉，下午返程。

20. 预算感受：
    可选：低 / 中等 / 中高 / 高

21. 你觉得最好的游玩月份：
    示例：5月、6-8月、9-10月、冬季雪后

## 资料与联系方式

22. 参考资料链接：
    可填写政府官网、文旅官网、景区官网、地图链接、游记链接等。

23. 推荐人昵称：
    可选。

24. 是否愿意后续补充信息：
    可选：愿意 / 不愿意

25. 联系方式：
    可选，微信/邮箱均可。公开展示前会去除个人联系方式。

## 审核标准

- 优先县城、乡镇、村落、街道或景区片区，不收集纯城市商圈。
- 目的地需要有明确旅游资源，不只是“好吃”或“离得近”。
- 推荐理由要说明为什么适合放松、短假或特定季节。
- 交通建议要尽量具体，但实时班次以后续查询为准。
- 内容审核后再加入配置文件，避免重复、失真或过度商业化。
